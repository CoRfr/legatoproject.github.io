Setup Target Software

Your target device should have come with a working Legato framework installed on it already.

To update the application and framework binaries on your target with the ones you built in Build Legato, use the update tool to pass the system update pack to the target device for installation.

$ update build/wp85/system.wp85.update 192.168.1.2

See update for more information on how to use the update tool.
