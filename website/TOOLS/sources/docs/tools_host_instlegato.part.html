instlegato

NAME

instlegato - install sample Legato system on target

SYNOPSIS

instlegato BUILD_DIR [DEST_IP]

DESCRIPTION

Once Legato has been built, instlegato can be used to install its sample 'system' on the target at the IP address specified by DEST_IP. BUILD_DIR is the path to the directory to which Legato was built.

If the target Legato directory is in the normal build location, i.e. Inside $LEGATO_ROOT/build, then you only need to specify the target platform, e.g. 'wp85', otherwise a full path is required.

If DEST_IP is not specified on the command line, the DEST_IP environment variable will be used, if it is set.
