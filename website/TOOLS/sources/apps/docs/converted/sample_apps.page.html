All sample apps are located in apps/sample/. The built apps are located in build//bin/samples.

Start App

To start an app, run this on the target:

root@swi-mdm9x15:~# app start

To stop an app on the target, run: 

root@swi-mdm9x15:~# app stop

Run app help for options.
