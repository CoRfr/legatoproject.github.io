GPIO for CF3

This prototype sample demonstrates a CF3 GPIO app using the GPIO Service. It uses adef executables, bindings, and processes sections, and cdef requires, cflag, and sources sections.

Run this to install the sample app:

$ cd apps/sample/gpioCf3Demo
$ make wp85
$ instapp gpioCf3Demo.wp85
