fwupdate

NAME

fwupdate - download or query modem firmware

SYNOPSIS

fwupdate help
fwupdate download FILE [TARGET_IP]
fwupdate query [TARGET_IP]

DESCRIPTION

fwupdate help

  • Print this help message and exit

fwupdate download FILE [TARGET_IP]

  • Download the given CWE file. After a successful download, the modem will reset.

fwupdate query [TARGET_IP]

  • Query the current firmware version. This includes the modem firmware version, the bootloader version, and the linux kernel version. This can be used after a download and modem reset, to confirm the firmware version.

ENVIRONMENT

If the TARGET_IP value is not given and the environment variable DEST_IP
is set then DEST_IP will be used as TARGET_IP
