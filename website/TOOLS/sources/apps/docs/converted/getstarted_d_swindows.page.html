Windows Host

Legato can be installed on a Windows host to use the Developer Studio IDE.

Legato ADK Installer

Download the Legato Application Development Kit (ADK) Installer for Windows from the Sierra Wireless Source Legato Downloads page.

Run the Legato ADK Installer and follow the steps to install the Legato ADK:

These items will be installed:

  • Developer Studio
  • drivers
  • latest Legato version

Cheat Sheet

After installation completes, launch Developer Studio, and follow the provided interactive cheat sheet to run your first Legato Hello World application on your target.
