startapp

NAME

startapp - start an app on target

SYNOPSIS

startapp APP_NAME [DEST_IP]

DESCRIPTION

If the application APP_NAME is installed on the target at DEST_IP then the app will be started.

If the APP_NAME is given as "*" (including the quotes) then all apps on the target will be started.

If DEST_IP is not specified on the command line, the DEST_IP environment variable will be used (if set).
