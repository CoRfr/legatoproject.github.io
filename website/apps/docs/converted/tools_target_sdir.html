    Use the Service Directory tool to interact with the Service Directory to control IPC bindings and do troubleshooting.

    Usage

    sdir [OPTION]

    sdir list

    list command generates a list of all the IPC services known by the Service Directory including servers advertising, users waiting for servers to advertise, and IPC bindings in effect.

    sdir load

    load command updates the Service Directory's bindings to match the bindingConfig settings found in the system configuration tree.
