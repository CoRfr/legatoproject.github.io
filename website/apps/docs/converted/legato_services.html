    This section contains detailed info about Legato's ready-built services available for commonly needed features. The Services Overview page has high-level info.

    This table lists Legato's Services — multi-app safe means arbitration is fully managed among apps so multiple apps can share the same resources.

    Service Description multi-app safe
    AirVantage AirVantage server for communication and data
    Audio Control audio features
    Cellular Network Register and manage modems
    Config Tree Target configuration database
    Data Connection Request data connection
    GPIO Configure general purpose input/output
    Modem modem services
    Positioning posititioning services
    Power Manager Device power management
    Secure Storage Secure storage for sensitive data
    SMS SMS and SMS Inbox
    Supervisor Control framework and apps
    Update update modem firmware and apps
    Voice Call Service Voice communications
